# Markdown

HTML from Markdown following [GitHub Flavored Markdown](https://help.github.com/articles/github-flavored-markdown/) syntax can be styled easily using `.gb-markdown`.

{% example html %}
<div class="gb-markdown">
    <h1>Heading 1</h1>
    <h2>Heading 2</h2>
    <h3>Heading 3</h3>
    <h4>Heading 4</h4>
</div>
{% endexample %}

# Popovers

Add small overlay content, like those found in iOS, to any element for housing secondary information.

{% example html %}
<div class="popover">
    <div class="popover-arrow"></div>
    <div class="popover-heading">
        <h4>Title</h4>
    </div>
    <div class="popover-body">
        Body of the modal
    </div>
</div>
{% endexample %}
